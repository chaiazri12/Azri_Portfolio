## Project 
### Capstone Project - Telecom Churn Data

**Code:** ['capstone_github.ipynb'](https://github.com/chaiazri12/Azri_Portfolio/blob/409fbd1a2bc8f1ee2d0e355924269defff7776bc/Capstone_Github.ipynb)
